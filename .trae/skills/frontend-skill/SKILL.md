# Frontend Skill (Premium Art Direction)

Use this skill when the quality of the work depends on art direction, hierarchy, restraint, imagery, and motion. Default toward award-level composition.

## Working Model
Before building, define:
1. **Visual Thesis**: One sentence describing mood, material, and energy.
2. **Content Plan**: Hero, support, detail, final CTA.
3. **Interaction Thesis**: 2-3 motion ideas that change the feel of the page.

## Core Rules
- **No Cards by Default**: Use layout, columns, and whitespace instead of boxed containers.
- **Full-Bleed Heroes**: Treat the first viewport as a poster, edge-to-edge.
- **Typography First**: Brand name should be the loudest text. Keep copy short and scannable.
- **Motion with Purpose**: At least 2-3 intentional motions (entrance sequence, scroll-linked effects).

## Imagery
- Use strong, real-looking images. Prefer in-situ photography.
- Avoid abstract gradients or fake 3D objects as primary anchors.
